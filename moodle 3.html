HTML5

HTML is Hypertext and Markup Language

HTML5 is used for structuring and presenting content on the World Wide Web.
It is the fifth and final major HTML version that is a WWW Consortium recommendation.
The current specification is known as the HTML Living Standard.
It has introduced application programming interfaces (API) and Document Object Model (DOM).

FEATURES :
  • There are new graphics elements including vector graphics and tags.
  • Enrich semantic content by including header, footer, article, section and
    are added.
  • It has introduced new multimedia features which supports both audio and
    controls by using audio and video tags.
